# SMMProject

Структура проекта для работы с Comfy UI, воркфлоу и деплоем на VPS.

- `comfy_ui/` – функции и адаптеры для взаимодействия с Comfy UI.
- `workflows/` – готовые пайплайны (Python/Shell).
- `vps_templates/` – Docker‑Compose, systemd‑юниты и скрипт быстрого развёртывания.
- `scripts/` – утилиты вспомогательные.
